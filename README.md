## Deployment instructions
  ### Mac osx
  1. run `createuser -P -d vetapp` and when promted add vetapp as the password
  2. run 'echo `export vetapp_DATABASE_PASSWORD="vetapp"' >> ~/.zshrc`
  3. run `source ~/.zshrc`
